
## 🚀 Getting Started

1. Install dependencies:

   ```bash
   git clone https://github.com/phamnhungoctuan/confidential-bomb
   cd frontend
   npm install
   ```

3. Run dev server:

   ```bash
   npm run dev
   ```

4. Open dev server:

   ```bash
   http://localhost:5173/
   ```
